Prerequisites

Basic knowledge in using software tools for image analysis is helpful but not mandatory\n\n

Learning goals

  • Introduction into cell microscopy image analysis
  • Application of software tools for automated analysis of image data

Keywords

Computer-based image analysis, image preprocessing, segmentation, feature extraction, classification, colocalization, tracking\n\n
